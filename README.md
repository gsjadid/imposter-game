## ğŸ® What does this project do?
This application allows a group of players to join a shared game room from their own devices.
- **Real-time Lobby:** Players join instantly using a room code.
- **Immersive Experience:** A completely redesigned interface with glassmorphism, vibrant colors, and smooth animations.
- **Role Assignment:** One player is secretly assigned the "Imposter" role, while others are "Civilians" who know the secret word.
- **Game Phases:** Structured gameplay with Clue Giving, Discussion, and Voting phases, all guided by interactive UI elements.
- **Mobile-First Design:** Fully optimized for mobile devices with dynamic viewport sizing (`100dvh`) for a native-app feel.
