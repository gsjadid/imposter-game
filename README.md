## ğŸ® What does this project do?
This application allows a group of players to join a shared game room from their own devices.
- **Real-time Lobby:** Players join instantly using a room code.
- **Role Assignment:** One player is secretly assigned the "Imposter" role, while others are "Civilians" who know the secret word.
- **Game Phases:** Structured gameplay with Clue Giving, Discussion, and Voting phases.
- **Mobile-First Design:** Optimized for playing on phones while sitting together.
